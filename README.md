# Social-Lorem-Vue

Project create with Vue JS

A social media application where you can:

- Create a post
- Like a post
- Comment on a post
- Follow and unfollow users
- Search for a user
- Edit your profle
- Change password with email (local environment only)

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```
